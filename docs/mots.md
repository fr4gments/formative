# Vocabulaire IKAL

Cette page liste les mots retenus ou candidats pour le langage IKAL. Un mot n'est stable que s'il respecte trois conditions :

- la forme est valide New Ithkuil ;
- le sens lexical est documente ;
- l'effet IKAL est motive par ce sens.

## Statuts

| Statut | Signification |
| --- | --- |
| Retenu | Mot valide et candidat direct pour le langage IKAL. |
| Candidat | Mot interessant, mais mapping artistique encore a confirmer. |
| Transition | Mot POC conserve temporairement pour comparaison. |
